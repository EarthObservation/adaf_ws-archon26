# adaf_ws-archon26

ARCHON automatic detection of archaeological features workshop 2026
This repo contains the materials for a short (45 minute) workshop on ADAF. This course starts from the basics, and assumes you have no prior coding experience.


## Get started
To start, follow these steps:

* Go to Google Colab
* If needed, create a Google account and/or log in to your Google account
* You are now in a so called notebook, where you can run code
  * Run cells by pressing 'play' button, or press SHIFT+ENTER
  * Edit code in cells where needed
